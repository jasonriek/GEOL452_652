How to Install and Use the Geophysics Java Applets

  • Step 1:
    • First make sure you have installed BlueJ, get it here if you need to install it.
    • Once you have clicked on the download link a prompt should show up depending on which browser you're using (look at fig. 1 below).


      fig. 1



      *Note: For this example, I'm using "GravitySurvey.zip;" however, this will apply to all future applets that we use (e.g. MagneticSurvey.zip).


    • If you're using Chrome, then right click on the bottom tab in the browser and click "Open" (fig. 1). Then drag the folder out of the zip file "GravitySurvey.zip" and place "Gravity Survey" folder in your location of choice ( e.g. desktop).
    • If you're using Firefox, then a dialog will show up (fig. 1), if you choose "Open with" it will open the zip file, and you would do the same as the Chrome step, and place the "Gravity Survey" folder in your location of choice.

      However, if you decide to choose the "Save File" option, then you will have to go into your "downloads" folder and double click on the zip file "GravitySurvey.zip", and then drag the Gravity Survey folder in your location of choice.

  • Step 2:
    • Run BlueJ (in start menu) then once in BlueJ (fig.2) click "Project" then click open Project.



      fig. 2



    • Once in the "Project Open" window, navigate to where you placed the "Gravity Survey" folder, click inside the "Gravity Survey" folder, and open the "GRAV" file with a little BlueJ icon by it (fig. 3).



      fig. 2



    • BlueJ will then boot up applets, and you should see a window like the one below on your screen (fig. 4).



      fig. 4



  • Step 3:
    • Once BlueJ is open, right click (fig. 5) on "GravityTunnel" applet and then click "Compile." When clicking "Compile" a window saying, "show() in java.awt.Window has been deprecated," just click "close." The same steps will also need to be done to the "GravityVerticalShaft" applet. After both have been compiled, you won't have to do it again.



      fig. 5



  • Step 4:
    • Now right click again on the "GravityTunnel" applet and click "run" (fig. 6).



      fig. 6



    • After clicking run, a window should pop up (fig. 7). Everything should be fine in the window, so just proceed by clicking the "Ok" button. At this point you should see two windows (fig. 8) that make up the "GravityTunnel" applet.



      fig. 7





      fig. 8
