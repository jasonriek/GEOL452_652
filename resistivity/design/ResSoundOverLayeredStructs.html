Resistivity Modeling - Resistivity Soundings Over Layered Structures

This JavaScript program will produce synthetic resistivity soundings, both Wenner and Schlumberger, over horizontally layered structures. You can use this script to design a resistivity survey for the project described in the request for bid. The graphic shown on this web page is the sounding result for the survey designed in the Survey Frame over the structure described in the Model Frame. You can vary the earth model by clicking with the mouse on the layer boundaries and dragging them to new depths and by setting the resistivity of each layer using the slider bars at the bottom of the Model Frame.

Survey parameters; survey type, minimum electrode spacing (AB/2 for Schlumberger and a for Wenner), number of electrode spacings per decade in distance to collect, number of decades in distance over which to collect data, etc., are controlled via the slider bars in the Control Frame.
