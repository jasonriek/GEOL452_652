Refraction Seismic: Data Acquisition

Acquiring Your Data

Using the survey parameters you defined in the previous exercise you should now generate field observations for interpretation. Fill in the specified parameters in the form that you download below, and a data set will be generated for you. If your survey involves multiple soundings, simply use this form to generate data for each refraction profile. Do not submit the cost of each profile separately. Submit only the total survey cost and repeat this cost for each sounding. All location parameters must be specified in standard survey coordinates .

*For instructions on how to extract and use the acquisition form, click here.

X Coordinate of Source Location: Indicates the east-west position within the survey area of the source.

Y Coordinate of Source Location: Indicates the north-south position within the survey area of the source.

Source Type: Type of source you would like to use - dynamite in shot holes, Betsy Seisgun, or a sledgehammer.

Trend of Profile: Direction in which geophones will be laid, either north-south or east-west. We assume that the geophones and the source lie along a line in this direction.

X or Y Coordinate of First Receiver: This is the east-west position of the first receiver for and east-west trending line or the north-south position of the first receiver for a north-south trending line.

Receiver Spacing: This specifies the receiver spacing, either east-west for an east-west trending line or north-south for a north-south trending line. Receiver locations will start at the coordinates of the first location and be spaced to the north or to the east from this location.

Number of Receivers to Deploy:Number of receivers to be deployed from the initial receiver location.

Number of Shots to Stack: Number of sources, each located at the source location given above, to record and sum together for noise reduction. This parameter is only used if you have choosen a hammer as your source.

Total Survey Bid: Your bid for completing the survey as defined in the Request For Bid .
